Présentation du projet

TaskFlow est une application web de gestion de projets collaboratifs destinée aux équipes pédagogiques et aux étudiants.

Problématique

Les enseignants et étudiants manquent d'outils simples et centralisés pour suivre l'avancement des projets tutorés. TaskFlow répond à ce besoin avec une interface intuitive et des fonctionnalités adaptées.

Fonctionnalités principales

  • Création et gestion de projets
  • Attribution de tâches aux membres
  • Tableau Kanban interactif (glisser-déposer)
  • Dépôt et partage de fichiers
  • Notifications par email
  • Calendrier partagé

Technologies utilisées

  • Front-end : HTML5, CSS3, JavaScript (ES6+)
  • Back-end : PHP 8, MySQL
  • Versioning : Git, GitHub
  • Hébergement : AlwaysData / Netlify
  • Design : Figma, Canva
